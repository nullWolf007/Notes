# java的基本类型和引用类型

## 区分
基本类型分配在栈stack中，引用类型的数据分配在堆heap中，但是他的引用还是在stack中

## 基本类型(8种)
### 四种整型
* byte    1字节
* short   2字节
* int     4字节
* long    8字节
### 两种浮点类型
* float   4字节
* double  8字节
### 一种字符类型
* char    2字节
### 一种布尔类型
* boolean 1位

## 引用类型(3种)
### 类class
* Object
* String
* Date
* Void
* ······
### 接口Interface引用
* List<E>:列表
* Map<key,value>
* 等等
### 数组Array
